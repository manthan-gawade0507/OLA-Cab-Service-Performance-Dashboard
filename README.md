# 🚖 OLA Ride Operations Dashboard | Power BI, SQL & Excel

## 📌 Project Overview

The **OLA Ride Operations Dashboard** is a business intelligence project designed to analyze ride bookings, revenue, cancellations, customer ratings, and operational performance. The dashboard provides interactive visualizations that help monitor key business KPIs and support data-driven decision-making.

## 🎯 Objectives

* Analyze ride booking trends and operational performance.
* Track revenue, cancellations, and customer satisfaction.
* Monitor driver and customer activity through interactive dashboards.
* Provide actionable insights to improve business efficiency.

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard development & data visualization
* **SQL** – Data extraction and analysis
* **Microsoft Excel** – Data cleaning and preprocessing
* **DAX** – KPI calculations and interactive measures

## 📊 Key Features

* Interactive KPI cards for Bookings, Revenue, Cancellation Rate, and Ratings.
* Dynamic filters (slicers) for date, ride type, payment method, and location.
* Drill-through reports for detailed operational analysis.
* Data modeling and DAX measures for accurate business insights.
* Clean and user-friendly dashboard layout.

## 📈 Business Insights

* Identified booking and revenue trends across different time periods.
* Analyzed ride cancellation patterns to improve operational efficiency.
* Evaluated customer ratings to measure service quality.
* Delivered actionable insights to support strategic business decisions.

## 🚀 Outcome

This project demonstrates practical skills in **Power BI, SQL, Excel, Data Modeling, DAX, Data Visualization, and Business Intelligence**, enabling organizations to monitor ride operations and make informed decisions through interactive dashboards.
